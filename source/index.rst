Excel-to-JSON by WTSolutions, Excel add-in, Convert Excel to JSON within Excel (web online, windows, mac, linux)
=================================================================================================================
Excel to JSON by WTSolutions is a **Microsoft Excel add-in** or **Web Application** which can convert Excel to JSON within Excel. 

Both **flat and nested JSON** can be converted using Excel-to-JSON.


当前为英文版文档，中文版详见 https://excel-to-json.wtsolutions.cn/zh-cn/latest/
