# Announcement: Excel-to-JSON by WTSolutions Feature Adjustment Notice

Dear Excel-to-JSON by WTSolutions Users,

Hello! In order to better meet user needs and provide better services, we have decided to adjust the feature scope of Excel-to-JSON's free and paid versions. Here are the specific changes:

## Feature Adjustments

### Free Version Adjustments
- **Feature Expansion**: The free version can now use all features previously only available in the paid version, including but not limited to:
  - Convert all visible worksheets
  - Convert all worksheets
  - First column as header
  - Nested JSON structure conversion
  - Multiple nested JSON key delimiters (dot, underscore, double underscore, forward slash)
  - Convert empty cells to JSON null
  - Exclude empty cells from JSON
  - Convert boolean values to strings "true"/"false"
  - Convert boolean values to numbers 1/0
  - Convert date formats to ISO 8601 strings
  - Output format as 2D array
  - Single object JSON output as object {}
- **Usage Limit**: The free version can convert a maximum of 6 rows of data per conversion

### Paid Version Adjustments
- **Feature Retention**: The paid version can still use all advanced features
- **Usage Limit**: The paid version has no row limit and can convert any amount of data
- **Other Benefits**: Paid version users still enjoy an ad-free experience

## Effective Date

This adjustment will officially take effect on March 14, 2026.

## Impact on Users

- **Free Version Users**: You will be able to use more advanced features, but the number of data rows converted per conversion will be limited to 6 rows
- **Paid Version Users**: Your user experience will remain unchanged, and you can still use all features without restrictions
